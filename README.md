# PlaylistFilter

A web application that allows users to paste a Spotify playlist URL, extract genre tags from each track's artist, let the user filter by genre, and create a new Spotify playlist in their account with only the matching tracks.

## Folder Structure

- `frontend/`: Frontend code for the application.
- `backend/`: Backend service for handling Spotify API interactions and business logic.
